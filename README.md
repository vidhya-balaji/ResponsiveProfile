Responsive Profile Card
This project demonstrates how to create a responsive profile card using HTML, Tailwind CSS, and JavaScript. The card adjusts seamlessly to different screen sizes, making it an ideal layout for personal or professional profiles on both mobile and desktop views.

Features
Responsive Design: The profile card adjusts to screen sizes from mobile to desktop using Tailwind CSS utility classes for flexible layouts.
Profile Picture: Easily customizable image section for the user's avatar.
Basic Information: Displays essential details such as name, role, location, and social links.
Interactive Elements: Includes a hover effect and a modal popup for more information, powered by JavaScript.
Dark Mode: Simple toggle between light and dark themes.
Tech Stack
HTML: Used for the structure of the profile card.
Tailwind CSS: Provides a utility-first approach to styling, allowing quick and efficient layout adjustments.
JavaScript: Adds interactivity such as hover effects, dark mode toggle, and modal popups.
